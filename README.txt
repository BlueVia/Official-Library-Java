==============================
====== BlueVia Java SDK ======
==============================

FILE CONTENTS:

- target/bluevia_java_sdk-1.0-SNAPSHOT.jar: 		Library with Bluevia SDK.

- target/dependency: 					Directory with the dependencies needed to use the library bluevia_java_sdk-1.0-SNAPSHOT.jar 
							Developer must add these libraries to project classpath or Maven configuration file (see Developer Guide for more information).

- src/main/java:					Source Code. Includes examples (com.bluevia.examples package).

- doc: 							SDK documentation: installation guide, code examples and API reference.

- pom.xml:						Maven project configuration file.
