Detailed Description

Class representing the MmsMessageReq that will be send using the MMS Client API This type is composed of the following fields:

  • addressList: mandatory; max occurrences: minimum 1.
  • subject: optional; String
  • originAddress: mandatory; String

This implementation is not synchronized
