List<MessageURIType> com.telefonica.schemas.unica.rest.mms.v1.MessageReferenceType.getAttachmentURL ( )

Gets the value of the attachmentURL property.

This accessor method returns a reference to the live list, not a snapshot. Therefore any modification you make to the returned list will be present inside the JAXB object. This is why there is not a set method for the attachmentURL property.

For example, to add a new item, do as follows:
