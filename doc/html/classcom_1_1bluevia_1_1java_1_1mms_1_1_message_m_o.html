List<MessageReferenceType> com.bluevia.java.mms.MessageMO.getMessages ( String  registrationId,
boolean  useAttachmentIds 
) throws JAXBException, BlueviaException

Retrieves the list of received messages. Depending on the value of the useAttachmentsIds parameter, the response will include the IDs of the attachments or not. If the ids are retrieved, the function 'getAttachment' can be used; otherwise, the attachments must be obtained throught the getMessage function.

See also:
