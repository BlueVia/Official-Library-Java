Member Enumeration Documentation

enum com::bluevia::location::data::LocationInfo::ReportStatus

Enum representing the values of the status of a location retrieval. Possible values are:

  • RETRIEVED: The location has been retrieved
  • NON_RETRIEVED: The location has not been retrieved
  • ERROR: There was an error retrieving the location
