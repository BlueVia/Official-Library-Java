\subsection{Detailed Description}
This class implements a part of a Multipart post object that consists of a file. Two different types of files are supported: 
\begin{DoxyItemize}
\item text files, for the following content types: 
\begin{DoxyItemize}
\item text/plain 
\item application/xml 
\item application/json 
\end{DoxyItemize}
\item binary files, for the following content types: 
\begin{DoxyItemize}
\item image/jpeg 
\item image/bmp 
\item image/gif 
\item image/png 
\item audio/midi 
\item audio/wav 
\item audio/mpeg 
\item audio/amr 
\item video/avi 
\item video/x-\/ms-\/asf 
\item video/mpeg 
\item video/mpeg4 
\item video/3gpp 
\end{DoxyItemize}
\end{DoxyItemize}
