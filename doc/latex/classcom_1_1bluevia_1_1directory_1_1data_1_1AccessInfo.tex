\subsection{Detailed Description}
Class to hold the information of the User Access Information block resource from the gSDP This type is composed of the following fields: 
\begin{DoxyItemize}
\item Access Type: it indicates the access network used to get connected. Possible values are: GSM, GPRS, UMTS, HSPA, LTE, WIMAX, etc. 
\item APN: Access Point Name.  
\item Roaming: It indicates if the user is attached to an access network different from its home network. 
\end{DoxyItemize}
