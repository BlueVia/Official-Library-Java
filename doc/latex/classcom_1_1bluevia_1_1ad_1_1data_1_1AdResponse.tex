\subsection{Detailed Description}
Class to hold information about the advertisement data received from the gSDP It includes 
\begin{DoxyItemize}
\item \hyperlink{classcom_1_1bluevia_1_1ad_1_1data_1_1Ad}{Ad}: Element that includes information for ad (one, for correct responses). Note: only one may be present in \hyperlink{classcom_1_1bluevia_1_1ad_1_1data_1_1AdResponse}{AdResponse}. 
\item Id: Unique request identification (BlueVia appID + timestamp). 
\item Version: Current version of the advertising protocol. 
\end{DoxyItemize}
